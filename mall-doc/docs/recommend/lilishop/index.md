## 1.2、商城介绍
官网：https://pickmall.cn
Lilishop商城系统支持商家入驻，后端基于SpringBoot 研发，前端使用 Vue、uniapp开发， 系统全端全部代码开源
前后端分离，支持分布式部署，支持Docker，各个API独立，并且有独立的消费者。

## 1.3、商业授权
商业版本与开源版本代码一致，没有区分
商业使用需要授权，授权方式可选择联系官网客服，或者qq群联系群主。
商业授权模式为永久授权，支持永久升级。
商业案例由于涉及部分多层二开关系，如需了解可以咨询销售。
